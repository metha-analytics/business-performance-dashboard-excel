# 📊 Business Performance Dashboard

## Overview

This project presents an interactive business performance dashboard developed using Microsoft Excel.

The dashboard consolidates key business metrics from multiple operational areas into a single reporting interface, enabling stakeholders to monitor performance, identify trends, and support data-driven decision making.

---

## Business Objectives

- Monitor company KPIs
- Track operational performance
- Analyze website performance
- Monitor marketing activities
- Support executive reporting

---

## Dashboard Modules

- Executive Overview
- Website Performance
- Digital Advertising
- Mobile App Performance
- SEO Performance
- Social Media Performance

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Formulas
